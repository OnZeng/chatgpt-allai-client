# ChatGPT-allai-client

## 介绍

简约版 `ChatGPT` 后续会接入世界上所有 ai 接口，国内支付宝和微信支付,PayPal
前端`Nuxt3+Naive-ui`SSR 渲染，**项目上线后可在各大搜索引擎搜索到，欢迎 star**
后端`Nest.js+Mysql` 采用 javascript 编写全栈项目
此项目没有过度封装，小白都能看懂，主打简洁，可不断完善！
对于项目如有疑问，可加下面 `QQ` 群交流！

# Chat Client

基于 Vue 3 + Vite + Tailwind CSS 的聊天应用前端

## 技术栈

- Vue 3.5.13
- Vite 6.0.5
- Vue Router 4.5.0
- Pinia 2.2.6
- Tailwind CSS 3.4.17

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

应用将在 http://localhost:3000 启动

## 构建

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```

## 环境变量

创建 `.env` 文件（可选）：

```
VITE_API_BASE=http://localhost:3001
```

默认 API 地址为 `http://localhost:3001/api`
